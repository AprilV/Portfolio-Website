1. Hover Interactions (Micro-interactions)
Purpose: Subtle animations signal interactivity and polish.

Suggestions:

Card Elevation on Hover: Slight upward motion (transform: translateY(-4px);) with soft shadow transition (box-shadow) adds depth.

Tag Animation: Fade-in underline or soft background pulse when hovering over tags like Academic or 20+ Years.

CSS Example:

  box-shadow: 0 8px 20px rgba(0, 0, 0, 0.12);
  transition: all 0.3s ease;
}
